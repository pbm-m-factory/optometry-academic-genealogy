# Optometry Academic Genealogy

An interactive, evidence-led academic family tree for postgraduate research supervision in optometry and vision science.

The current preview includes 200 people and 255 named supervision relationships across PhD, MPhil, MSc, MScOptom, MASc, MAppSci, Masters Research, Ophthalmic Doctorate and DSc awards. Supporting sources are shown only as “Uploaded document” or “Public institutional record”. Filenames, authors, dates, page references, private source identifiers and local document paths are excluded.

Relationships are classified as Confirmed, Supported or Provisional. The website provides filters for institution, degree, year and confidence, a conservative duplicate-name review, and a public contribution form that prepares a reviewable GitHub issue.
