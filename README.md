# Optometry Academic Genealogy

An interactive, evidence-led academic family tree for postgraduate research supervision in optometry and vision science.

The current preview includes 125 people and 148 named supervision relationships across PhD, MPhil, MSc, MScOptom, MAppSci and DSc awards. Supporting sources are shown only as “Uploaded document” or “Public institutional record”. Filenames, authors, dates, page references, private source identifiers and local document paths are excluded.
